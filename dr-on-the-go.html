Description

Doctor on the Go is a telehealth app clone. It allows patients to schedule and conduct online consultations with doctors and allow doctors to prescribe medications.

Technologies Used

Frontend

EJS, Bootstrap

Backend

PostgreSQL, Node/Express

Cloud

GitHub, Heroku, AWS S3

Main Features

  • A patient can schedule/cancel a date to consult a doctor and view past consultation records.
  • A doctor can accept, start, cancel and end consultations and view past consultations.
  • Forum-like chatbox for patient and doctor to interaction during the consultation.
  • A doctor can prescribe medicine for the patient.
  • Patient and Doctors can update their profiles.

Main Learning Points

  • Learnt Express web framework and integrating with EJS template engine to render front-end pages.
  • Learnt SQL syntax to query the PostgreSQL relational database.
  • Learnt Bootstrap to style pages
