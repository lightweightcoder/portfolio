Description

Hangout? aims to connect new people with common interests and help them save money by leveraging group deals on activities.

Technologies Used

Frontend

React(Create React App), AJAX(Axios), Bootstrap

Backend

PostgreSQL/Sequelize, Node/Express

Cloud

GitHub, Heroku, Firebase storage, Netlify

Main Features

  • A user (the creator) can create/edit/delete an activity.
  • A user (the participant) can join/leave an activity.
  • Participants and the creator can chat once they join an activity.

Main Learning Points

  • Integrated 2 separate repositories into a full-stack application - 1 repository for frontend Create React App and 1 repository for backend Express/Node/Postgres.
  • Implemented a real-time messaging feature using Firebase cloud database (NoSQL).
  • Setting cookies across origins (backend to frontend server).
  • Implemented Deep-linking using React Router Dom npm package.
  • Implemented useContext and useReducer React hooks to share states among multiple levels of components without having to pass the states down as props.
