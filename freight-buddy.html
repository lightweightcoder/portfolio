Description

Freight Buddy was created to help people (requesters) gain access to their beloved overseas products by enlisting the help of others overseas to purchase the products and ship them over to the requesters.

Technologies Used

Frontend

React, AJAX(Axios), Bootstrap, Webpack

Backend

PostgreSQL/Sequelize, Node/Express

Cloud

GitHub, Heroku, AWS S3

Main Features

  • A user (requester) can create/cancel a request for a product.
  • A user (potential helper) can offer help by accepting a request.
  • A user can update the status of a request (eg. if the helper has bought the product or it has already been shipped and awaiting confirmation of receipt from the requester)

Main Learning Points

  • Integrated Webpack into React to pre-process and compile JS and CSS files to reduce file processing time.
  • Implemented React hooks to manage states.
  • Implemented Sequelize for ORM database querying to improve code robustness by reducing human errors.
  • Implemented Multer npm package to enable multi-file uploads of listing images to AWS S3.
  • Implemented Bootstrap cards and carousel components.
  • Architected application according to Model-View-Controller pattern to follow code organisation best practices.
